Zur Verstärkung unseres Teams in Köln suchen wir Sie als Softwareentwickler Java (m/w)

Ihre Aufgaben

  • Sie arbeiten mit bei der Anforderungsanalyse, erarbeiten gemeinsam Lösungswege und -vorschläge, schreiben Konzepte und realisieren diese abschließend in verschiedenen Technologien – hauptsächlich Java.
  • Sie betreuen und entwickeln bereits bestehende, komplexe Webanwendungen im Bereich der Online-Reisevermarktung weiter.
  • Sie halten durch Code-Reviews mit Ihren Teamkollegen die Qualität des Produkts auf einem hohen Niveau.
  • Sie erkennen Verbesserungspotentiale der Infrastruktur und optimieren diese kontinuierlich, um die Entwicklung noch effizienter zu gestalten.

Ihr Profil

  • Sie haben ein abgeschlossenes Studium der Informatik oder eine vergleichbare Qualifikation / Ausbildung und haben Erfahrung im Bereich der Java Anwendungsprogrammierung, insbesondere für Web-Applikationen.
  • Sie verfügen über breite Fachkenntnisse mit aktuellen Entwicklungsumgebungen, Frameworks/Tools und Webtechnologien.
  • Sie haben Erfahrung mit J2SE- und J2EE-Technologien (JSP, Java Servlets , Java Beans) sowie Java Bibliotheken, z.B. Spring, Struts, Hibernate/JPA, DOM/SAX.
  • Sie sind kommunikationsstark, teamfähig, eigenverantwortlich, können sich flexibel und schnell auf Neues einstellen und haben Spaß beim Entwickeln von Software.
  • Gute Linux-Kenntnisse sind von Vorteil.
  • Gute Deutschkenntnisse runden Ihr Profil ab.

Hierfür bieten wir Ihnen neben einem modernen Umfeld und herausfordernden Aufgaben
